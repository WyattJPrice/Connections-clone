# Connections-Clone

A close to exact clone of NYT Connections game.

This site became pretty popular at school after its release:

![Connections Stats](https://connections.wyattprice.dev/api/stats.svg?v=1)


## Features

- Light/dark mode toggle
- Daily Gamemode
- Custom mode, user can select or shuffle categories
- Admin Panel w/ stats, daily puzzle creator, community tab for revising
- Telegram and browser notifications for Admin on new categories submitted
- [Leaderboard](https://connections.wyattprice.dev/leaderboard) for categories created and played
- Mobile Friendly
